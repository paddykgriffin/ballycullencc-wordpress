== Description ==

Display your sermons from soundfaith.com on your WordPress website.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->SoundFaith Name screen to configure the plugin
4. Your account ID should be your church's Faithlife group ID. E.g. 6317013
5. Use the shortcode generator or WordPress visual editor button to add your sermons.
